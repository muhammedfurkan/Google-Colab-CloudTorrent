- [x] Simple Torrent, peerflix-server and deluge remote torrent client

- [x] rclone
- [x] Jdownloader
- [x] Youtube-dl
- [x] Extracte/Archive Zip/Tar/Rar
- [x] Netdata
- [x] Cloud Commander
- [x] Ssh and noVnc
